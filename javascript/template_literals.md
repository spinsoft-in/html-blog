###### Task 1: Append FirstName and LastName

``` js
    let firstName = "Naresh" ;
    let lastName = "Kumar";

    // Appending String
    let fullName = firstName + " " + lastName ;
    console.log(fullName);
    // Output: Naresh Kumar

    // Using Template Literals
    let result = `${firstName} ${lastName}` ;
    console.log(result);
   // Output: Naresh Kumar
```
